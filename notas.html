Registro de Cambios

Documentación técnica de actualizaciones y parches.

Versión 1.1.0 (Actual) Abril 2026
Integración de bibliografías y correcciones
  • Módulo de estudio: Creación e integración de tres secciones de bibliografía técnica para HTML, CSS y Bootstrap 5.
  • Motor de examen: Se implementó una función de limpieza de cadenas para permitir el renderizado de etiquetas HTML dentro de las preguntas y opciones sin romper el DOM.
  • Optimización UX: Refactorización del lenguaje de la interfaz para mayor legibilidad y tono profesional.
  • Corrección de color: Rediseño del sistema de retroalimentación visual en los resultados de examen para evitar falsos positivos de calificación.
  • Mini-juego interactivo: Reestructuración de la lógica de misiones mediante un sistema de capas superpuestas (Layer Detection) y sincronización con el nuevo panel de ayudas de comandos CSS.
Versión 1.0.0 Lanzamiento inicial
Despliegue inicial del simulador
  • Infraestructura básica: Menú principal basado en guías de estilo retro-tech.
  • Base de datos: Implementación de 240 preguntas originales divididas por niveles (Junior, Mid, Senior) y categorías temáticas.
  • Simulador interactivo: Desarrollo inicial del prototipo de entrenamiento visual con Flexbox (Modo Estrella).
  • Compatibilidad: Integración de librerías externas de Bootstrap y Animate.css.
