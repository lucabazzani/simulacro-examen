Métodos de Planificación

Técnicas para planificar, programar y controlar las actividades del proyecto.

1. Método PERT

PERT (Técnica de Revisión y Evaluación de Programas): Ideal para proyectos con incertidumbre en los tiempos.

Se utiliza para programar detalladamente las tareas y crear una hoja de ruta visual que facilite la aprobación directiva.

Pasos para crear un diagrama PERT:
Paso 1: Identificar las tareas del proyecto (Fase inicial).
Paso 2: Definir las dependencias (Desglose del trabajo).
Paso 3: Vincular las tareas (Comprensión de los participantes).
Paso 4: Establecer el cronograma (Tiempo de duración).
Paso 5: Gestionar el progreso (Proceso de control de cambios).
2. Método de la Ruta Crítica (CPM)

CPM (Critical Path Method): Diseñado para proyectos con tiempos de actividad conocidos.

  • Ruta Crítica: Identifica la secuencia más larga de actividades que deben finalizarse a tiempo para completar el proyecto.
  • Priorización: Permite gestionar recursos de forma efectiva y evitar obstáculos ejecutando solo una tarea a la vez por dicha ruta.
  • Cálculo de Holgura: Determina cuánto puede retrasarse una tarea no crítica sin afectar la fecha final.
Importante: Si una de las tareas de la cadena crítica se termina con retraso, afectará directamente la duración de TODO el proyecto.
3. Tipos de Representación en Red
Método de Precedencia (DPR / MDP)

También llamado Red de Nodos. Representa las actividades como nodos (cajas) y las flechas para ilustrar las relaciones de dependencia. Es altamente flexible.

Método del Diagrama de Flechas (DFA)

Aquí las flechas representan las actividades y los nodos representan eventos o hitos de control.

4. Diagramas de Gantt

Gráfico de barras que muestra la duración de las actividades a lo largo del tiempo. Permite apreciar todas las fechas clave de un solo vistazo.

  • Barras: Representan una etapa o tarea; su longitud es la duración.
  • Hitos: Representan puntos de verificación y momentos importantes (Verificación de entregables).
  • Línea de base: Facilita el seguimiento de los plazos comparando el progreso real contra el estimado.
