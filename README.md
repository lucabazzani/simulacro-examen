# Simulador de Examen

Este proyecto es una plataforma de práctica para estudiantes de desarrollo web. Permite evaluar conocimientos técnicos sobre HTML5, CSS3 y Bootstrap 5 mediante exámenes dinámicos y desafíos de código en tiempo real.

## Características principales

- Tres niveles de dificultad: inicial, intermedio y avanzado.
- Doce variantes de exámenes: distribuidos por temas específicos o integrales.
- Banco de 240 preguntas: basadas en documentación técnica oficial.
- Modo interactivo: sistema de entrenamiento para diseño con Flexbox con actualización visual instantánea.
- Interfaz adaptable: diseño minimalista y técnico enfocado en la usabilidad.

## Tecnologías utilizadas

- HTML5
- CSS3 (Variables y Flexbox)
- Bootstrap 5
- JavaScript (Vanilla JS / Módulos ES6)
- Animate.css (Efectos de transición)

## Estructura del proyecto

- index.html: menú de selección principal.
- examen.html: motor de evaluación dinámica.
- estrella.html: entorno de juego interactivo.
- js/preguntas.js: base de datos indexada de preguntas.
- js/engine.js: lógica de procesamiento y corrección.
- js/estrella.js: controlador del modo interactivo.
- css/style.css: sistema de estilos global.

## Uso local

Para utilizar el simulador de forma local, clona este repositorio o descarga los archivos y abre el archivo index.html en cualquier navegador moderno. Al utilizar módulos de JavaScript, se recomienda abrir el proyecto mediante un servidor local como Live Server.
